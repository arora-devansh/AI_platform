# AI Platform

An AI-powered mock interview platform that helps users prepare for technical and behavioral interviews through interactive voice-based AI interview sessions.

## 🚀 Features

- 🤖 AI-powered mock interviews
- 🎤 Real-time voice conversations
- 🔐 Secure user authentication
- 📊 Interview history and progress tracking
- 📱 Responsive and modern UI
- ⚡ Fast performance with Next.js
- 🔥 Firebase backend integration

## 🛠 Tech Stack

- Next.js 15
- TypeScript
- React
- Tailwind CSS
- Firebase
- Vapi AI

## 📂 Project Structure

```
app/
components/
constants/
firebase/
lib/
public/
types/
```

## ⚙️ Installation

```bash
git clone https://github.com/arora-devansh/AI_platform.git
cd AI_platform
npm install
npm run dev
```

## 🔑 Environment Variables

Create a `.env.local` file in the project root.

```env
NEXT_PUBLIC_FIREBASE_API_KEY=
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=
NEXT_PUBLIC_FIREBASE_PROJECT_ID=
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=
NEXT_PUBLIC_FIREBASE_APP_ID=

NEXT_PUBLIC_VAPI_PUBLIC_KEY=
VAPI_PRIVATE_KEY=
GOOGLE_GENERATIVE_AI_API_KEY=
```
